# ft_transcendence

This is a browser-based, real-time multiplayer 3D web application built with Three.js for **ft_transcendence**, the final project at 42 Wolfsburg.

Users can join and enter a shared interactive world where they can see and interact with other players in real time. The system supports multiple connected clients — Just open the webpage and you’ll enter the shared world.

All players are synced in real time using a WebSocket-based system. A lightweight server handles incoming connections and player synchronization.

This is still a work in progress and not finished yet — it represents a preview of the system currently being developed.
